## Installation

### Step 1: Clone the Repository
```bash
git clone https://github.com/noobsmoker/quick-summarize-content-20260507_125337.git
cd quick-summarize-content-20260507_125337
```

### Step 2: Make the Script Executable
```bash
chmod +x summarize.py
```

### Step 3: Verify Installation
```bash
python summarize.py --help
```

### Step 4: (Optional) Add to System PATH
```bash
# For Unix/Linux/macOS - add to ~/.bashrc or ~/.zshrc
export PATH="$PATH:$(pwd)"

# For Windows - add to system environment variables
set PATH=%PATH%;%cd%
```
